## Kết quả đạt được tuần 5

- Hoàn thiện thiết kế kiến trúc hạ tầng dự án và đề xuất các mẫu tham khảo phù hợp cho nhóm.
- Xây dựng thành công sườn code (code skeleton) cùng các file cấu hình ban đầu, đảm bảo nền tảng vững chắc cho dự án.
- Đăng ký và kích hoạt tài khoản AWS Skill Builder, bắt đầu làm quen với các khóa học và tài liệu.
- Xác định nguyên nhân gây chi phí bất thường: tài nguyên EC2 không được xóa sạch và tài khoản người dùng thiếu kiểm soát, từ đó đề xuất các biện pháp tối ưu hóa chi phí.
- Tích hợp thành công Axios và fetch API để xử lý HTTP requests trong React.
- Nắm được cách xử lý các vấn đề CORS và áp dụng xử lý lỗi đúng cách cho API requests.
